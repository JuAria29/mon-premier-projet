Ce qu'il faut construire

Les 4 blocs techniques pour rendre Aria collaboratif.

1
Auth multi-utilisateurs
Invitation par email (Supabase Auth)
Attribution d'un rôle à la création du compte
Profil lié à un espace entreprise
2
Rôles personnalisables
Table `roles` + table `role_permissions` dans Supabase
Interface éditeur de permissions (matrice cochable)
Assignation d'un rôle à chaque utilisateur invité
3
Row Level Security (RLS)
Policies Supabase pilotées par le rôle de l'utilisateur
Un commercial ne voit que ses lignes — au niveau DB
Dirigeant : aucun filtre
4
Dashboards conditionnels
Navigation filtrée selon le rôle connecté
Composants affichés / masqués selon les permissions
Dashboard mobile-first pour Technicien & Assist. Tech.
